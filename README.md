# Concejo Municipal de Distrito de Nosara

Sitio informativo sobre la propuesta de un **Concejo Municipal de Distrito (CMD)** para el distrito de Nosara, Guanacaste, Costa Rica.

Incluye qué es un CMD, marco legal, cifras del distrito, cronograma, preguntas frecuentes y un formulario para sumar apoyos. Disponible en **español e inglés**, con **modo claro y oscuro**.

**Sitio:** https://egutierrezcr.github.io/nosaraconcejomunicipal/

> Documento informativo y de participación ciudadana. Las piezas de propuesta (presupuesto, organigrama, reglamento, plan de desarrollo, simulador) son **modelos hipotéticos** para validación de concepto; no son documentos oficiales.
